Brainfuck Toolkit
-----------------

To learn about Brainfuck see the [Wikipedia page](https://en.wikipedia.org/wiki/Brainfuck).

This project is just a toy that containts:

- a Brainfuck compiler
- a BF source code printer
- a Ruby DSL to write BF source code in Ruby

## Example

This is how you write the famous [Fizz Buzz](https://en.wikipedia.org/wiki/Fizz_buzz) problem with our DSL:

```ruby
Generator.new.bf do
  is_nothing = byte
  fizz = alloc(4, 'fizz')
  buzz = alloc(4, 'buzz')
  nl =  byte("\n")

  times(100) do |idx|
    set(is_nothing, 1)

    callz(mod(idx, 3)) do
      print_arr(fizz)
      zero(is_nothing)
    end

    callz(mod(idx, 5)) do
      print_arr(buzz)
      zero(is_nothing)
    end

    callnz(is_nothing) { print_decimal(idx) }
    print(nl)
  end
end
```
